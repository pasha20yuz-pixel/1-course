# Лабораторная работа №1: Исследование линейных конгруэнтных генераторов (ЛКГ)

**Дисциплина:** Структуры данных  
**Семестр:** 2  

## Цель работы
Разработать программу на языке C для работы с ЛКГ, реализующую команды подбора параметров, генерации последовательностей и статистического тестирования. Исследовать зависимость периода от параметров и качество генерируемых последовательностей с помощью критерия Пирсона.

## Содержание папки
- `lcg.c` – основная программа (обработка команд `get_c`, `get_a`, `lcg`, `test`)
- `lcg_period.c` / `lcg_period.h` – функция вычисления периода (алгоритм Флойда)
- `period_tool.c` – утилита для автоматического определения периода по набору параметров
- `input.txt` – пример входной команды для `lcg`
- `params.txt` – тестовые наборы параметров для `period_tool`
- `Лабораторная_СД_1.docx` – отчёт с теоретическими сведениями, таблицами и выводами

## Сборка и запуск

### Требования
- Компилятор C99 (gcc, clang, MSVC)
- ОС: Windows / Linux / macOS (для Windows рекомендуется использовать MinGW или WSL)

### Компиляция основной программы (lcg)
```bash
gcc -std=c99 lcg.c -o lcg.exe
Компиляция утилиты для определения периода
bash
gcc -std=c99 lcg_period.c period_tool.c -o period_tool.exe
Запуск
Основная программа читает команду из файла input.txt и выводит результат в output.txt.
Пример содержимого input.txt:

text
lcg a=5 x0=1 c=5 m=7776 n=5000
Запуск:

bash
./lcg.exe
Результат появится в output.txt.

Утилита period_tool читает параметры из файла или stdin. Пример:

bash
# Одиночный запуск
./period_tool 8 1 7 101

# Пакетный режим из файла params.txt
./period_tool -f params.txt

# Чтение из stdin
./period_tool -
Примеры использования
1. Поиск всех c, взаимно простых с m
В input.txt:

text
get_c cmin=1 cmax=100 m=7776
Вывод – список чисел.

2. Поиск минимального a, удовлетворяющего теореме Кнута
В input.txt:

text
get_a m=7776
Вывод: 13

3. Генерация последовательности
В input.txt:

text
lcg a=13 x0=1 c=5 m=7776 n=100
Вывод – 100 чисел (по одному на строке) в output.txt.

4. Тест на равномерность (χ²)
В input.txt:

text
test inp=sequence.txt
Вывод – статистика χ² и вердикт.

Результаты исследования (кратко)
Для параметров, удовлетворяющих теореме (a=13, c=5, m=7776), период максимален (7776) и распределение равномерно.

Нарушение условий (a-1 не делится на все простые делители m, либо c не взаимно просто с m) уменьшает период, но равномерность может сохраняться.

Критерий Пирсона подтверждает, что даже при не максимальном периоде последовательность может быть статистически случайной.

Подробнее – в отчёте Лабораторная_СД_1.docx.

Примечания
Программа корректно обрабатывает 64-битные числа (до 2^64-1).

Для больших m возможно переполнение при умножении в формуле ЛКГ – в коде предусмотрена защита (возврат 0).

Статистический тест использует адаптивное разбиение на интервалы с ожидаемой частотой ≥5.

Оптимизация (рекомендации)
В функции find_period алгоритм Флойда уже оптимален по памяти O(1).

При генерации длинных последовательностей компилируйте с -O2.

Для миллиардов чисел используйте буферизованный вывод.

Лицензия
Учебный код, свободное использование в образовательных целях.
