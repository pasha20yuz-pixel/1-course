# Лабораторная работа №1: Исследование линейных конгруэнтных генераторов (ЛКГ)

**Дисциплина:** Структуры danych  
**Семестр:** 2  

## Цель работы
Разработать программу на языке C для работы с ЛКГ, реализующую команды подбора параметров, генерации последовательностей и статистического тестирования. Исследовать зависимость периода от параметров и качество генерируемых последовательностей с помощью критерия Пирсона.

## Содержание папки
- lcg.c – основная программа (обработка команд get_c, get_a, lcg, 	est)
- lcg_period.c / lcg_period.h – функция вычисления периода (алгоритм Флойда)
- period_tool.c – утилита для автоматического определения периода по набору параметров
- input.txt – пример входной команды для lcg
- params.txt – тестовые наборы параметров для period_tool
- Лабораторная_СД_1.docx – отчёт с теоретическими сведениями, таблицами и выводами

## Сборка и запуск

### Требования
- Компилятор C99 (gcc, clang, MSVC)
- ОС: Windows / Linux / macOS (для Windows рекомендуется MinGW или WSL)

### Компиляция основной программы (lcg)
`ash
gcc -std=c99 lcg.c -o lcg.exe
Компиляция утилиты для определения периода
bash
gcc -std=c99 lcg_period.c period_tool.c -o period_tool.exe
Запуск
Основная программа читает команду из файла input.txt и выводит результат в output.txt.
Пример input.txt:

text
lcg a=5 x0=1 c=5 m=7776 n=5000
Запуск:

bash
./lcg.exe
Результат появится в output.txt.

Утилита period_tool:

bash
./period_tool 8 1 7 101           # одиночный запуск
./period_tool -f params.txt       # пакетный режим
./period_tool -                   # чтение из stdin
Примеры использования
get_c – поиск взаимно простых c

get_a – минимальное a по теореме Кнута (для m=7776 → 13)

lcg – генерация последовательности

test – критерий χ² для проверки равномерности

Результаты исследования (кратко)
Параметры, удовлетворяющие теореме (a=13, c=5, m=7776), дают максимальный период (7776) и равномерное распределение.

Нарушение условий уменьшает период, но равномерность может сохраняться.

Критерий Пирсона подтверждает статистическую случайность.

Подробнее – в отчёте Лабораторная_СД_1.docx.

Примечания
64-битные числа, защита от переполнения в find_period.

Адаптивное разбиение на интервалы в тесте χ² (ожидаемая частота ≥5).

Оптимизация
Для больших n компилируйте с -O2.

Алгоритм Флойда оптимален по памяти (O(1)).

Лицензия
Учебный код, свободное использование в образовательных целях.
