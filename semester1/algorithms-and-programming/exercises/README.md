# Упражнения по курсу «Алгоритмы и программирование»

Здесь собраны небольшие самостоятельные задачи на C.

## Структура
- strings/ – работа со строками
  - _to_x – замена 'A' на 'X'
  - everse – переворот строки
  - everse_words – переворот слов в предложении
- rrays/ – одномерные массивы
  - lphabet – сортировка букв
  - 	en_numbers – сумма, произведение, мин/макс, ближайшее к среднему
  - 	hree_massive – генерация трёх массивов, поиск минимума/максимума
- ile_io/ – ввод/вывод из файлов
  - count_length_lines – подсчёт длины строк
  - count_symbols – поиск самого частого символа
- math/ – математические алгоритмы
  - compound_interest – сложные проценты
  - simple_divisors – разложение на простые множители
  - 	riangle_pascal – треугольник Паскаля
- conditionals/ – ветвления
  - information_for_user – возраст, выпуск, пенсия, воинская обязанность
  - x_and_y – числа от 10 до 3*max(x,y)
- itwise/
  - work_with_bytes – побитовые операции

## Сборка и запуск
Откройте любой .c файл в VS Code и нажмите F5 (универсальная конфигурация в корне репозитория).
