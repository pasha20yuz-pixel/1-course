# Алгоритмы и программирование (C)

## Лабораторные работы
- [lab-03-magic-square](./labs/lab-03-magic-square/) – магический квадрат 3×3
- [lab-04-sorting](./labs/lab-04-sorting/) – сравнение алгоритмов сортировки (быстрая, Шелла, выбором, шейкерная)
- [lab-06-hashtable](./labs/lab-06-hashtable/) – хэш-таблица с цепочками (отдельная папка, см. labs/lab-06-hashtable)

## Упражнения
exercises/ – небольшие задачи по темам: строки, массивы, файловый ввод/вывод, математика, условные операторы, битовые операции.
Подробнее в [exercises/README.md](./exercises/README.md)

## Сборка
- Для отдельных .c файлов используйте универсальную конфигурацию .vscode в корне репозитория (нажмите F5).
- Для многофайловых проектов смотрите локальные .vscode или Makefile.
